# 🎵 Application de génération de musique Jazz
**Projet Python — Interface graphique + génération algorithmique MIDI**

---

## 📌 Présentation
Cette application permet de générer automatiquement des mélodies de style jazz grâce à des algorithmes simples et interactifs.  

Elle propose une interface graphique permettant de choisir des paramètres musicaux, de lancer la génération et d’écouter le résultat en MIDI.

Ce projet a été réalisé dans le cadre d’un travail universitaire visant à explorer :

- la génération algorithmique de musique
- la manipulation du MIDI
- la création d’interfaces graphiques en Python

---

## 🚀 Fonctionnalités
- Interface graphique intuitive (Tkinter)
- Génération aléatoire de notes jazz (gammes, motifs, variations)
- Lecture MIDI en temps réel
- Gestion du tempo et des durées
- Sauvegarde des séquences générées
- Architecture modulaire (fonctions séparées pour interface, génération, lecture)

---

## 🛠️ Technologies utilisées
- Python  
- Tkinter — interface graphique  
- Pygame / Pygame.midi — génération et lecture MIDI  
- Random — génération aléatoire des notes  
- Time — gestion du tempo  
- OS — gestion des fichiers  

Architecture modulaire (≈ 260 lignes de code)

## 📦 Installation

Cloner le dépôt :
    git clone https://github.com/nynyBao/Application-de-conception-musique-jazz-.git

Installer les dépendances :
    pip install pygame

Lancer l’application :
    python "Application_jazz code.py"

---

## 🎧 Utilisation
1. Ouvrir l’application  
2. Choisir les paramètres (tempo, longueur, etc.)  
3. Cliquer sur **Générer**  
4. Écouter la mélodie générée  
5. Sauvegarder si souhaité
